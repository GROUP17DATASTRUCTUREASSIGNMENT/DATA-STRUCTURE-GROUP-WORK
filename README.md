# Queue and Singly Linked List Implementation

## Course

BIT8122 – Data Structures and Algorithms

## Programming Language

Python

## Description

This project implements the following data structures from scratch as required by the assignment:

### Task 3: Queue

The Queue is implemented without using Python's built-in queue module.

Operations implemented:

Enqueue
Dequeue
Peek

The program demonstrates a sequence of queue operations and prints the queue after each operation.

Time Complexity

| Operation | Complexity |
| --------- | ---------- |
| Enqueue   | O(1)       |
| Dequeue   | O(n)       |
| Peek      | O(1)       |

---

### Task 4: Singly Linked List

A Singly Linked List is implemented using custom `Node` and `LinkedList` classes.

Operations implemented:

 Insert at Head
 Insert at Tail
 Delete a Node by Value
 Traverse and Print the List

Time Complexity

| Operation       | Complexity |
| --------------- | ---------- |
| Insert at Head  | O(1)       |
| Insert at Tail  | O(n)       |
| Delete by Value | O(n)       |
| Traverse        | O(n)       |
